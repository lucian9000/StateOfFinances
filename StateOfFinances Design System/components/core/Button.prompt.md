Pill-shaped action button for all CTAs in the StateOfFinances dashboard.

```jsx
<Button variant="primary">Add Transaction</Button>
<Button variant="danger" size="sm">Delete</Button>
<Button variant="income">Confirm Income ↑</Button>
<Button variant="ghost" disabled>Unavailable</Button>
```

**Variants:** `primary` (violet fill), `secondary` (surface-raised), `ghost` (transparent + text-muted), `danger` (magenta tint), `income` (cyan tint).  
**Sizes:** `sm`, `md`, `lg`. Use `fullWidth` for form submit buttons.
